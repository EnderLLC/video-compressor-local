# Task Tabanlı Çalışma Sistemi

Her task tamamlandıktan sonra `tasks/archive/` klasörüne taşınır.

## Nasıl Çalışır?

1. Yeni bir task oluşturulur: `tasks/TASK-XX-DESCRIPTION.md`
2. Task aktif olarak `DURUM.md` dosyasında belirtilir.
3. Task tamamlanınca, dosya `tasks/archive/` altına taşınır.
4. `DURUM.md` güncellenir ve yeni task aktif hale getirilir.
